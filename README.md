## Prerequisites

* [Python 2.7](https://www.python.org/downloads/)
* [ZeroMQ](http://zeromq.org/bindings:python)

  ```py
  pip install pyzmq
  ```
* [Node.js LTS](https://nodejs.org/en/)


## Installation
For Python 2.7 do:
```bash
python setup.py install
```
For Node:
```bash
npm install
```

## Running
Running the python module:
```
python handoff.py
```
Running the Node module:
```
node index.js
```
